## OverLoading / OverRiding

> OverLoading

    같은 클래스내에서 같은 이름의 메서드를 사용하는 것
	    * 매개변수 타입이 달라야한다.
	    * 매개변수 개수가 달라야한다.
	    
> OverRiding

    부모클래스에서 정의한 메서드를 자식 클래스에서 변경하는 것
	     * Static 메서드는 오버라이딩을 허용하지 않음
